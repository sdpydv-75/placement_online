📌 Product Requirements Document (PRD)
Project Name: Online Placement & Internship Portal
1. 📖 Overview

Yeh platform students ko internships aur job opportunities find karne me help karega aur companies ko candidates hire karne ka system provide karega.

2. 🎯 Objectives
Students ko centralized portal provide karna
Companies ko hiring process simplify karna
Admin ko full control dena (users, jobs, reports)
3. 👥 User Roles
1. Student
Register/Login
Profile create/update
Resume upload
Jobs/Internships search
Apply to jobs
Application status track
2. Recruiter / Company
Register/Login
Company profile manage
Job/Internship post karna
Applications review karna
Candidates shortlist/reject
3. Admin
User management
Job moderation
Reports & analytics
Platform control
4. 🔑 Core Features
Authentication
JWT based login/signup
Role-based access
Student Features
Profile dashboard
Resume upload (PDF)
Job search (filters: location, skills, type)
Apply system
Recruiter Features
Job posting form
Applicant tracking system
Candidate filtering
Admin Features
Dashboard
Manage users
Manage jobs
Reports
5. 📊 Functional Requirements
Feature	Description
User Auth	Secure login/signup
Job Listing	CRUD operations
Application System	Students apply to jobs
Notifications	Email / in-app
Search Filters	Skills, location
6. 🚫 Non-Functional Requirements
Secure (JWT, bcrypt)
Scalable (MongoDB)
Responsive UI
Fast API response
7. 📈 Future Scope
AI Resume Matching
Interview scheduling
Chat system
Recommendation engine
8. 🧪 Success Metrics
Active users
Applications per job
Recruiter engagement
